# File Converters

There are a range of different file converters in Haystack that
can extract text from files and cast them into the unified Document format

|||
|-------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
|__Position in a Pipeline__| At the very beginning of an indexing Pipeline |
|__Input__       | Files                                                                                                                                                                  |
|__Output__      | Documents or dictionaries                                                                                                             |
|__Classes__     | PDFToTextConverter, DocxToTextConverter, AzureConverter, ImageToTextConverter, MarkdownConverter                                                                                                           |
|||

Haystack also has a `convert_files_to_dicts()` utility function that
will convert all txt or pdf files in a given directory.

```
from haystack.utils import convert_files_to_dicts
docs = convert_files_to_dicts(dir_path=doc_dir)
```
